07 DocumentApi

这个示例聚焦展示宿主如何取回编辑中的画板内容。你可以先载入一份演示文件，再在 iframe 内继续编辑，随后回到宿主侧读取当前内容，确认拿到的是最新的结构化数据。

Embedded Editor baseUrl = http://localhost:8080 · appId = 123456

Document 操作

宿主通过 SDK 先注入一份演示内容，再主动读取当前画板状态。这样无论用户在 iframe 里如何继续编辑，宿主都能拿到最新数据，用于同步、调试或后续处理。
