08 SlidesApi

这个示例专门演示编辑模式下的 SlidesApi。宿主侧通过 SDK 直接切换、新增、插入、复制、删除和重排画板，并实时读取当前文档，方便联调桥接协议与画板状态同步。

快捷操作

这些按钮基于当前画板索引进行操作，适合快速走通主流程。

任意索引操作

用于测试 switchTo、addAt、duplicate、remove、reorder 的自定义索引输入。

switchTo(index)

addAt(index)

duplicate(index, targetIndex?)

remove(index)

reorder(fromIndex, toIndex)

文档快照

每次操作后都会重新读取一次 editor.document.getContent()，这里展示当前画板结构，便于确认多页顺序与数据内容。

提示：如果要验证切页效果，可以在 iframe 内手动给某一页增加元素，再用宿主侧切换不同页观察内容变化。
