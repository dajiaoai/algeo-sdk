Algeo SDK 示例

选择以下案例查看内嵌画板的不同用法。

演示模式

01 基础嵌入

100% 宽度、居中 max-width、vh、双列等高频布局

02 SDK 方式

使用 AlgeoSdk 封装 iframe 与 postMessage

03 切换画板

多画板内容，通过 switchSlide 切换

04 loadFile

输入 JSON 文本或导入 .json 文件加载内容

05 REPL

执行 REPL 指令，获取面向 AI 的文档/文本内容

编辑模式

06 内嵌编辑模式

预接入 editor mode，验证新 URL 构造与初始化参数

07 DocumentApi

设置初始画板内容，以及如何取回当前画板内容

08 SlidesApi

编辑模式下的切换、新增、复制、删除与重排画板能力

09 HistoryApi

测试撤销、重做、跳转与清空历史记录的宿主调用能力
